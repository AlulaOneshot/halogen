# phosphor GUI

A simple custom GUI framework in rust. It uses taffy for flex layout, and skia-safe for rendering. I decided to build this because flutter is very much designed for mobile application and cupertino/material UI, and I needed something that fit my use case.

| Platform | Arch                             | Tested OS Version | Stability |
|----------|----------------------------------|-------------------|-----------|
| macOS    | Apple Silicon (arm64)            | 26.3.1 (a)        | Full      |
| macOS    | Apple Silicon (x86_64 emulation) | 26.3.1 (a)        | Full      |

More platforms will be released as I can test them (soon).
